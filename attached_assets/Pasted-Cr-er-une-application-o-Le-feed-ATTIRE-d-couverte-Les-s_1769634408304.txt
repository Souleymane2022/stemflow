3. FEED MULTI-FORMAT : Entité Content (video/text_post/image_post/quiz/infographic) avec ContentCard adaptatif, badges salon d'origine, bouton "Rejoindre salon"

4. PERSONNALISATION IA : engagement_stats par catégorie, algorithme scoring (intérêts 25%, difficulté 30%, engagement 25%, diversité 15%, streak 5%), preferred_difficulty auto-ajusté

5. SALONS RENFORCÉS : Rôles (apprenant/challenger/mentor/modérateur), publications variées, défis internes, leaderboard, connexion bidirectionnelle avec feed

6. MISSIONS DYNAMIQUES : Entité Mission (daily/weekly/one_time), génération auto personnalisée selon profil/intérêts/salons, types (watch/quiz/join/comment/share/battle/create/streak)

7. MÉTRIQUES : VideoEngagement avec watch_time/completion/interactions, utiliser pour améliorer (pas pénaliser)
